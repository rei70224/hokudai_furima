# 北大フリマ（仮）
北大生同士でメルカリっぽく教科書などをやり取りできるサービスを目指します。

## 特徴
- 北大生だけで安全な売り買いができるフリマサービス
- 直接手渡しで渡すスタイルなので、運命の出会いも・・・？
